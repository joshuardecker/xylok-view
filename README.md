# Xylok View

A desktop application for browsing and reviewing DISA Security Technical Implementation Guides (STIGs). Load a STIG file, inspect individual rules and their findings, and track compliance status across your checklist.

**Supported formats:** XCCDF v1.1, CKL, CKLB, and Xylok internal.

# How to Build and Run the Application
To run the application:
- Install rustc and cargo
- Run ```cargo run --release``` in the base directory of this project
